
A responsive Netflix landing page UI clone built using HTML, CSS. This project replicates the look and feel of Netflix’s homepage with modern design, smooth layout, and interactive elements.

Tech Stack :
HTML5 – Structure of the website
CSS3 – Styling and responsiveness
JavaScript – FAQ toggle functionality


Project Structure : 

Netflix-UI-Clone/
│
├── index.html
├── style.css
├── assets/
│   ├── images/
│   ├── icons/
│   └── favicon.ico
└── README.md

How It Works : 
The homepage layout is built using Flexbox and Grid
The FAQ section uses JavaScript to toggle answers dynamically
The Trending section uses horizontal scrolling for better UX
CSS media queries ensure responsiveness across devices
